我的第一篇博客文章

##第一章
node常用命令

nodemon + 文件名 执行命令可以边修改代码边查看运行结果

npm + install + 第三方模块名 命令 可以下载安装第三方模块 -g代表全局安装

npm + uninstall + 第三方模块名 命令 可以卸载第三方模块

nrm ls 查询可用下载地址名称

nrm use 下载地址名称 切换下载地址

##第二章

gulp中提供的方法
gulp.src()获取任务处理的文件
gulp.dest()输出文件
gulp.task()建立gulp任务
gulp.watch():监控文件的变化

gulp插件

gulp-htmlmin :html 文件压缩
gulp-csso :压缩css
gulp-babel: JavaScript语法转换
gulp-less: less语法转换
gulp-uglify:压缩混淆JavaScript
gulp-file-include公共文件包含
browsersync 浏览器实现同步

插件的使用步骤 1.下载 2.引用 3.调用

快速生成package.json的命令 npm init -y

拿到package.json文件后 可以通过npm install 命令下载所需模块
